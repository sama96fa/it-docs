# Connexion au VPN

## Prérequis
- Compte pro + Internet
- Client VPN installé
- MFA disponible (si activé)

## Étapes
1. Ouvrir le client VPN.
2. Entrer `vpn.entreprise.com` (si demandé).
3. Se connecter avec email + mot de passe.
4. Valider la MFA.
5. Vérifier le statut **Connecté**.

## Dépannage
- **Identifiants refusés** : vérifier email complet, mot de passe expiré/compte verrouillé → Support IT.
- **MFA échoue** : heure du téléphone en automatique, refaire un code → Support IT si besoin.
- **Connecté mais pas d’accès interne** : déconnecter/reconnecter, redémarrer.
- **Time-out** : vérifier Internet, tester un autre réseau → Support IT.
